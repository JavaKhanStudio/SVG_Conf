SVG <use> + CSS Variables

Un SVG peut definir des elements reutilisables avec <defs> et <use>. Combinez cela avec des variables CSS pour creer des composants personnalisables.

Le pattern d'une app IA : ce que tu vois ici est exactement la recette qu'on retrouve dans une application augmentee a l'IA : un template SVG (le os, sa forme) plus des parametres (couleurs, texte) qui peuvent venir d'un humain... ou d'un LLM.
