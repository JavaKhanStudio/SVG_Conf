Le SVG, un outil sous-estime

Le SVG est un outil precieux pour n'importe quel developpeur web. Optimise pour toutes les tailles d'ecran, il offre une puissance et une elegance pixel-perfect. Mais pas seulement : d'une flexibilite incomparable, il est capable de s'adapter a vos utilisateurs avec un minimum de JavaScript.

Seul probleme...

Sa complexite peut effrayer (jetez un oeil a l'anatomie d'un path ou aux courbes de Bezier) :

SVG <path d="M30,45 C30,45 10,30 10,18 C10,12 14,8 20,8 C24,8 27,10 30,13 C33,10 36,8 40,8 C46,8 50,12 50,18 C50,30 30,45 30,45 Z" fill="#e74c3c"/>

Cette suite peut sembler abstraite pour un humain, mais heureusement, pas pour une IA — explorez l'atelier ou la galerie !

Venez decouvrir ou redecouvrir ce format d'image, dans une courte presentation avec de multiples exemples, dont certains generes en live !
