      Quand une IA utilise Studio

      Studio expose un vrai pipeline pour qu'un agent IA dessine un SVG parametrique a partir d'une description (pas d'une photo). Tout est decrit dans .claude/skills/svg-from-scratch/SKILL.md : l'agent ecrit dans studio-work/, renvoie un lien charge automatiquement, et itere sur les _Vn que tu sauves.

      1
      Brief

      Tu decris le sujet (« un lapin diabolique »), le format (icon, carte), le style. L'agent pose une question s'il en a vraiment besoin — une seule.

      2
      Scaffold v1

      L'agent ecrit studio-work/<base>.svg — silhouette + palette, sparse volontairement. Jamais directement dans images/ ou sources/.

      3
      Handoff

      Reponse avec le deep link studio.html?path=./studio-work/<base>.svg. Studio auto-charge, pas besoin de taper le chemin.

      4
      Tu ajustes

      Drag les anchors bleus et controles violets des paths, tweak les :root vars a droite, masque des elements via 👁, connecte / deconnecte une couleur a une variable.

      5
      Save (Vx)

      Clic sur Save ecrit <base>_V1.svg, puis _V2, etc. Versionnage auto, pas de perte, annulation par fichier.

      6
      L'agent reprend

      L'agent lit ton _Vn, voit tes ajustements, ecrit une v(n+1) en ajoutant un niveau de detail. Il attend ton Save avant d'iterer — pas de boucle autonome.

      7
      Animer

      Animations = @keyframes parametres par des :root vars (duration, easing, intensite). Le panneau :root expose les knobs automatiquement, le bouton ⏸ fige tout pour screenshotter.

      8
      Promotion

      Quand c'est bon : deplacement de studio-work/ vers la vraie destination — images/<cat>/ pour la galerie, sources/ pour une reference d'histoire, gallery/ pour Black Room.

      Le pattern : l'agent dessine, tu corriges visuellement, l'agent apprend tes corrections. Pas de Photoshop, pas de pinceau : l'interface est un inspector d'attributs et tes Saves sont le feedback. Source complet : .claude/skills/svg-from-scratch/SKILL.md.
