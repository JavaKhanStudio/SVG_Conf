Animations SVG

Decouvrez les differentes techniques d'animation SVG

Animation CSS

Les elements SVG peuvent etre animes avec les @keyframes CSS.

@keyframes spin { from { transform: rotate(0deg); } to { transform: rotate(360deg); } } rect { animation: spin 3s linear infinite; }

Pulse

Scale + transform-origin

@keyframes pulse { 0%,100% { transform: scale(1); } 50% { transform: scale(1.15); } }

Color shift

fill transition

@keyframes colorShift { 0% { fill: #BA0101; } 33% { fill: #0752CF; } 66% { fill: #2ecc71; } }

Animation SMIL

Animer directement dans le SVG avec <animate>.

<circle cx="30" cy="50" r="15"> <animate attributeName="cx" from="30" to="170" dur="2s" repeatCount="indefinite" /> </circle>

Animation de trait

Effet de dessin progressif avec stroke-dashoffset.

stroke-dasharray: 300; stroke-dashoffset: 300; animation: dash 3s ease forwards; @keyframes dash { to { stroke-dashoffset: 0; } }

Morphing de forme

Transformer une forme en une autre avec animate.

<path> <animate attributeName="d" dur="4s" values="triangle; square; circle; triangle" repeatCount="indefinite" /> </path>

Transformations

Rotation, scale, translate animees.

<!-- Each square has a different animation --> <rect style="animation: spin 4s..." /> <rect style="animation: bounce 1.5s..." /> <rect style="animation: pulse 2s..." /> <rect style="animation: fadeInOut 2s..." />

Animation interactive

SVG + JavaScript pour des animations reactives.

svg.addEventListener('click', () => { circle.setAttribute('r', Math.random() * 50 + 20); });

<animateTransform>

SMIL rotation native

<animateTransform attributeName="transform" type="rotate" from="0 50 50" to="360 50 50" dur="5s" />
