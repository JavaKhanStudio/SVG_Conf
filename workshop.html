Comment ca marche

Chaque SVG de la galerie declare ses parametres dans une regle :root, et accole un commentaire /* @ws ... */ qui dit a l'atelier comment construire le bon controle. C'est tout. Le fichier reste un SVG valide ouvrable dans n'importe quel navigateur.

CSS dans le SVG :root { --bg-color: #1a1a2e; /* @ws color */ --shape-size: 50; /* @ws number min=10 max=100 */ --accent: #f09040; --accent-shadow: #7a4820; /* @ws mix=accent:-0.55 */ }
