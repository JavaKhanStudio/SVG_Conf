Anatomie d'un SVG

Comprendre la structure du code SVG

Structure de base

Un SVG est un document XML avec un element racine <svg> et des elements enfants qui decrivent les formes.

SVG <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 200 200" width="200" height="200"> <!-- Shapes go here --> <circle cx="100" cy="100" r="80" fill="#0752CF" /> </svg>

Le ViewBox

L'attribut viewBox definit le systeme de coordonnees interne du SVG. Il permet de creer des graphiques responsifs qui s'adaptent a n'importe quelle taille.

viewBox viewBox="min-x min-y width height" <!-- Example: viewBox="0 0 100 100" --> <!-- Creates a 100x100 coordinate space --> <!-- The SVG scales to fit its container -->

Formes de base

SVG offre plusieurs formes primitives pour construire vos graphiques.

<rect>

<rect x="10" y="10" width="100" height="60" rx="5" />

<circle>

<circle cx="50" cy="50" r="40" />

<ellipse>

<ellipse cx="60" cy="40" rx="50" ry="30" />

<line>

<line x1="10" y1="10" x2="110" y2="70" stroke="#f39c12" />

<polygon>

<polygon points="60,5 110,40 90,95 30,95 10,40" />

<path>

<path d="M30,45 C30,45 10,30 10,18..." />

La balise Path

L'element <path> est le plus puissant du SVG. Il peut dessiner n'importe quelle forme avec des commandes de deplacement.

Commandes principales : M (deplacer), L (ligne), H (horizontal), V (vertical), C (courbe cubique), Q (courbe quadratique), A (arc), Z (fermer)

Stylisation

Les elements SVG peuvent etre styles avec des attributs, du CSS inline, ou des feuilles de style externes.

Attributes <rect fill="#0752CF" stroke="#BA0101" stroke-width="3" opacity="0.8" rx="10" /> <!-- Or with CSS --> <style> rect { fill: #0752CF; stroke: #BA0101; transition: fill 0.3s; } rect:hover { fill: #BA0101; } </style>
