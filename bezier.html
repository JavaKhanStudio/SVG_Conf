Les commandes d'un path

Toutes les lettres utilisables dans l'attribut d d'un <path>. En majuscule = coordonnees absolues, en minuscule = coordonnees relatives au point courant.

M x,y

Move to

Pose le stylo sans tracer.

L x,y

Line to

Trace une ligne droite.

H x

Horizontal

Ligne horizontale (y inchange).

V y

Vertical

Ligne verticale (x inchange).

Q cpx,cpy x,y

Quadratic

1 point de controle + fin.

T x,y

Smooth quad

Chaine une Q (CP reflete).

C c1 c2 x,y

Cubic

2 points de controle + fin.

S c2 x,y

Smooth cubic

Chaine une C (CP1 reflete).

A rx ry ... x,y

Arc elliptique

Arc d'ellipse (7 parametres).

Z

Close path

Revient au dernier M (ferme).

Astuce : les minuscules m l h v q t c s a z font la meme chose en relatif au point courant.

Chaine multi-segments (C + C)

Deux cubiques chainees se partagent un point de jonction. Glisse n'importe quel point : depart, controles de la premiere courbe, jonction centrale, controles de la seconde courbe, arrivee. C'est exactement comme ca que l'IA construit une fleur entiere : segment par segment.

Une courbe ecrite par une IA

Voici la fleur en cloche d'une heather, dessinee par Claude. Chaque ligne SVG ci-dessous est un element de l'image. Decoche une case pour voir ce qui disparait : la tige, la corolle, les nervures ou les etamines.

Chaque Q = un point de controle + un point d'arrivee. Sept commandes, une fleur. C'est ca qui rend les SVGs si interessants pour l'IA : sortie courte, structure claire, modifiable a la main. Tu pourrais ouvrir ce fichier et changer une couleur, deplacer un petale, ou ajouter une etamine, sans regenerer quoi que ce soit. Voir d'autres exemples dans la galerie.
